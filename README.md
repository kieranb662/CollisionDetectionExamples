#  SwiftUI Collision Detection Examples 

This is an example project to go along with the collision functions from the [CGExtender](https://github.com/kieranb662/CGExtender) package. 

## Collision Functions

* Point/Line
* Point/Circle
* Point/Rectangle
* Point/Polygon
* Line/Line
* Line/Rectangle
* Line/Circle
* Line/Polygon
* Circle/Circle
* Circle/Rectangle
* Circle/Polygon
* Rectangle/Rectangle
* Rectangle/Polygon
* Polygon/Polygon
